# Тестовое задание | fuse8

### Описание:

Веб-приложение, которое по запросу пользователя получает данные о выбранном персонаже с помощью API. Приложение имеет адаптивный интерфейс.

### Используемые технологии:

> - **HTML**
> - **SCSS**
> - **TypeScript**
> - **React**
> - **Vite**
